Started Project

Template d'un project simple en HTML, CSS {tailwind}, JavaScript et Angular Js

Étapes à suivre pour lancer le projet :

  • Changer le nom par défaut du projet "started_project" en faisant une recherche globale sur votre éditeur de code et en le remplacant par le nom de votre projet.

  • Assurer vous d'avoir lancer votre logiciel de serveur local (wamp, mampp, xampp...).

  • Lancer les commandes npm run tailwind pour démarer tailwind et gulp pour lancer le serveur.

  • Vous êtes prêt à coder

Configuration de base :

  • - Nom de l'application angular : ng-app="started_project"
  • - Nom du controller angular : ng-controller="ProjectBackController"
